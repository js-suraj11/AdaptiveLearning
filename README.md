# Adaptive Learning App
This is  an platform for teaching concepts to forgetful learners using adaptive teaching policy.[1] 

# Install the dependencies and Run 
<pre>
Clone the repository
</pre>
<pre>
$ pip install -r requirements.txt
</pre>
<pre>
$ python app.py
</pre>
